18.- Un árbol general lleno de grado 4, tiene 21 nodos.
a) ¿Cuál es la altura del árbol?
b) Desarrolle el proceso realizado para obtener la respuesta anterior

a) La altura es 2.
b) 	(4^(h+1)-1) / (4 - 1) = 21
	(4^(h+1)-1) / 3 = 21
	((4^(h+1)-1) / 3)* 3 = 21 * 3 // cancelo el denominador 3 con el 3 que estoy multiplicando.
	4^(h+1)-1 = 63
	4^(h+1) = 64
	4^(h+1) = 4^3
	h + 1 = 3
	h = 3 - 1
	h = 2
